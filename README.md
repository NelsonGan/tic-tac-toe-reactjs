## Description
Tic Tac Toe Minimax is an interactive web application that showcases the importance of algorithm optimization in game playing. With the use of Alpha-Beta Pruning, the app is equipped with an AI opponent that always selects the best optimal move in a match. The app also features a real-time display of the number of states explored in each move, highlighting the significant performance gains achieved with the optimization technique.

## Tech Stack
The application was built with ReactJS.
